display_history

PURPOSE

Display the object's history field in a friendly way

SYNOPSIS

function [] = display_history(subj,objtype,objname)

DESCRIPTION 

 Display the object's history field in a friendly way

 [] = DISPLAY_HISTORY(SUBJ,OBJTYPE,OBJNAME)

 You can use this to display the SUBJ's own history if OBJTYPE ==
 'subj' and OBJNAME == ''
